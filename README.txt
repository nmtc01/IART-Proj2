O nosso projeto foi todo desenvolvido utilizando Jupyter Notebook, ou seja, para a sua execução e compalição basta correr o comando "jupiter-notebook" numa janela de terminal na root do projeto.

Todo o projeto já se encontra compilado, mas pode ser testado e compilado com os butoes run do Jupiter. Não se recomenda no entanto correr o plot dos gráficos dos nossos datsets, assim como os testes de accurancies nos algoritmos de redes neuronais. 

Para compilação do projeto será também necessário ter python instalado no computador (recomendando versão 3) e as bibliotecas numpy, seaborn, matplotlib, pandas, scikit-learn e graphviz.
